# Step 03: ジェットカット（無音自動カット）

素材動画から無音部分を自動検出し、カットした動画を生成する。

## 前提条件
- Step 02（素材チェック）でメイン動画の存在・情報を確認済み
- FFmpegがインストール済み

## やること

### 0. 言い直しの事前確認

ユーザーに確認する：
**「動画内の言い直し部分は事前に削除してありますか？」**

- **削除済み** → そのまま次へ
- **未削除 / わからない** → ジェットカット前に不要部分を削除してから再撮影、または手動で削除してもらうよう伝えて次へ

### 1. 無音区間の検出

FFmpegの `silencedetect` フィルターで無音区間を検出する。

```bash
ffmpeg -i public/video/元動画.mp4 -af silencedetect=noise=-30dB:d=0.3 -f null - 2>&1 | grep "silence_"
```

パラメータ：
- `noise=-30dB` — 無音と判定する音量閾値（調整可能）
- `d=0.3` — 最低0.3秒以上の無音をカット対象にする（デフォルト）

### 2. カットリストの生成

検出結果から「残す区間（非無音セグメント）」のリストを作成する。

各セグメントの前後に **0.05秒のパディング** を入れて、カットが不自然にならないようにする。

### 3. カット動画の生成

FFmpegの `concat` フィルターまたは `select` フィルターで非無音部分だけを繋いだ動画を生成する。

```bash
# セグメントごとにカットして結合
ffmpeg -i public/video/元動画.mp4 -filter_complex "..." -c:v libx264 -c:a aac public/video/動画名_cut.mp4
```

出力ファイル名: `public/video/元ファイル名_cut.mp4`

### 4. 結果の確認

- カット前後の動画の長さを比較表示する
- カットされた無音区間の数と合計秒数を表示する
- カット率（何%短くなったか）を表示する

### 5. 元動画の保持

**元動画は削除しない。** カット版を別名で保存する。
以降のステップではカット版の動画を使用する。

## パラメータ調整ガイド

| 状況 | noise | d |
|------|-------|---|
| 標準（推奨） | -30dB | 0.3s |
| テンポ重視（きつめ） | -25dB | 0.2s |
| BGMあり・環境音多め | -40dB | 0.8s |
| ゆるめ（間を残す） | -30dB | 0.5s |

ユーザーの動画の特性に応じて調整する。結果を見て閾値を微調整することもある。

## 完了条件
- カット済み動画（`*_cut.mp4`）が `public/video/` に存在する
- カット前後の長さ・カット率を把握している
- 元動画が保持されている
