更新履歴

naoki-blueprint のバージョンアップ履歴（Semantic Versioning）

追加された機能・改善の一覧です。新しい動画を作るときは bash 新規作成.sh が自動で最新版を取得します。手動で更新したい場合は bash アップデート.sh を実行してください。

v1.10.0 2026-04-25
✨ 強調テロップが もっとカッコよくなりました

動画の冒頭やキーワードで使う 2種類の強調テロップ(赤の emphasis / 金の emphasis2)を細かく調整しました。次に作る動画から自動で適用されます。

  • 赤の強調テロップ(emphasis)に立体感が出ました: 文字の上下が濃い赤、中央が明るい赤になり、奥行きが見えるようになりました。
  • 赤の強調テロップに黒い影がはっきり入りました: 背景が明るい動画でも文字が浮き上がって見えやすくなります。
  • 2つとも斜体になりました: 文字に動きが出て、Hook の勢いが強まります。
  • 金の強調テロップ(emphasis2)がスッキリしました: 不要に重たかった外周を削って、金属感を残したまま読みやすく。
📌 あなたが やることはありません
  • 次回 ./アップデート.sh で自動反映。
  • 既に作った動画は見た目変わりません(再レンダしない限り)。新しい動画から自動で新デザインに。
v1.9.0 2026-04-25
✨ 「自分の話し方」を1回書けば、毎回反映されるようになりました

動画を作るたびに「私の一人称は◯◯で、口癖は△△で…」と説明する必要がなくなりました。1回書いておけば、台本生成・テロップ作成のすべてに自動で反映されます。

  • あなた専用のノート置き場が新設されました: my-workspace/ フォルダに「話し方ノート」「過去の台本」「お手本動画のメモ」を置けます。
  • 毎回の入力作業がなくなります: 一人称・口癖・NGワード・キラーフレーズ・テロップ量・絵文字の好みを一度書けば、新しい動画でも自動で読み込まれます。
  • 過去台本やお手本動画も活かせます: past-scripts/ や benchmark-videos/ にメモしておけば、新企画作りで自動参考にされます。
  • 使い方: my-workspace/my-style.example.md をコピーして my-style.md として埋めるだけ。
  • アップデートで上書きされない: 書いた内容はあなたのものとして守られます。
v1.8.0 2026-04-24
🔥 強調テロップが プロデザイン級にクオリティアップ

動画の Hook やキーワードに使う 2種類の強調テロップを、Photoshop で作ったような重厚な質感に作り直しました。サムネ並みの押し出し感が出ます。

  • 赤の強調テロップ(emphasis)が PSD級の質感に: 派手な赤グラデーション + 金色のグロー + 立体的な縁取りで、動画の冒頭が一気に映えます。
  • 金の強調テロップ(emphasis2)が金属反射に進化: 中心で反射が割れる本物の金属のような質感。数字や強調ワードがリッチに。
  • 使い方は今まで通り: text に文字を渡すだけ。書式の変更はゼロ。
  • emphasis3 は廃止: emphasis2 の新デザインに統合されたため、Python での事前生成は不要になりました。
v1.7.0 2026-04-24
🔥 カット後の発話の「ブツ切れ」が直りました

「あなた」の最初の「あ」が消える、「〜ね」の語尾が切れる、といった発話末尾・先頭が削られる問題を解消しました。次回作る動画から自然な発話に戻ります。

  • 発話の余韻が自然に残るようになりました: カット時に文字単位で詰めるのをやめ、前後にちょっと余白を残す方式に戻しました。
  • 動画の尺は少し長くなる(2分48秒の素材で +約3秒)が、聞こえ方は明らかに自然になります。
  • 既存プロジェクトはそのままでOK: すでにカット済みの動画は再カット不要。次回新規動画から自動で新方式に。
