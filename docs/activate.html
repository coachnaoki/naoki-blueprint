01環境セットアップ（初回のみ）

動画制作に必要な4ツールをインストールします。すでに入っている方はスキップしてOK。

macOS (Apple Silicon)
brew install node python@3.12 ffmpeg pip3.12 install mlx-whisper opencv-python npm install -g @anthropic-ai/claude-code
Windows

まずブラウザから事前準備:

  • Node.js: nodejs.org
  • Python 3.12: python.org（「Add Python to PATH」必須）
  • FFmpeg: gyan.dev/ffmpeg（PATHに追加）

その後 PowerShell で:

pip install faster-whisper opencv-python npm install -g @anthropic-ai/claude-code
⚠ Python は 3.12 限定。3.14 では mlx-whisper が動きません。

02リポジトリ取得 → Claude Code 起動

さん、準備を始めましょう。

⚠ 既に Claude Code を開いている方は、ターミナル上で Ctrl+C（Mac は Control+C）を 2回押して終了してから進めてください。

Cursor のターミナルで、naoki-blueprint を置きたいフォルダに移動（例: デスクトップ上の Cursor フォルダ）:

macOS
cd ~/Desktop/Cursor
Windows (PowerShell)
cd ~\Desktop\Cursor

そのフォルダでリポジトリをダウンロード（Mac / Windows 共通）:

git clone https://github.com/coachnaoki/naoki-blueprint.git cd naoki-blueprint

続けて Claude Code を起動:

claude --dangerously-skip-permissions

起動後は Claude に「この手順を進めて」と伝えるだけでも、自分で以下を順にターミナルで打っても OK。

03ライセンス認証

テンプレートフォルダ内で認証します。あなたのライセンスIDを貼り付け:

cd .template node scripts/validateLicense.mjs NK-XXXX-XXXX-XXXX cd ..

✅ 認証成功で .license ファイルが生成され、以降は自動認証されます。PCは1台に紐付きます。

04動画タイプを選んでプロジェクト作成

naoki-blueprint フォルダで ./新規作成.sh を実行します:

./新規作成.sh

起動すると、以下を順に聞かれます:

  1. プロジェクト名（日付でOK。例: 2026-04-19）
  2. 動画タイプ
1

YouTube 横動画

1920×1080 / 10〜20分
20ステップ / スライド・ワイプ・OP・ハイライト対応

2

ショート動画

1080×1920 / 30〜60秒
14ステップ / テロップ8種に絞り情報密度を優先

フォルダ作成・npm install まで自動。完了後に案内されるコマンドで Claude Code を起動してください。

05素材配置 → 作業開始

プロジェクトフォルダに移動:

cd projects/2026-04-19

素材を public/ フォルダに配置します（Finder / エクスプローラーからドラッグ＆ドロップでOK）:

public/ ├── videos/ 動画素材 │ ├── main/ 本編動画 ★必須 │ ├── inserts/ 物理挿入動画（任意） │ ├── overlays/ オーバーレイ動画（任意） │ ├── opening/ OP動画（横動画のみ・任意） │ └── highlight/ ハイライト動画（自動生成・横動画のみ） ├── images/ 画像素材 │ ├── inserts/ 横に表示する画像（任意） │ └── overlays/ 全画面表示画像・アニメーション付（任意） ├── bgm/ BGM（MP3） ★必須 ├── se/ SE効果音 ★必須 │ ├── 強調/ │ ├── ポジティブ/ │ └── ネガティブ/ ├── script/ ナレーション台本（推奨） └── output/ 完成動画の出力先

素材を置いたら、Claude Code で作業開始:

/step01-context

あとは AI が次のコマンドを案内してくれるので、質問に答えながら進めるだけ。

UPアップデート（バージョンアップ）

テンプレートやスキルが更新されたら、以下のコマンドで最新版に置き換えできます。
あなたが作った projects/ と .license は安全です。

naoki-blueprint フォルダで実行:

cd ~/Desktop/Cursor/naoki-blueprint ./アップデート.sh

Windows (PowerShell) の場合:

cd ~\Desktop\Cursor\naoki-blueprint bash アップデート.sh

⚠ アップデート中は Claude Code を Ctrl+C を 2回押して終了してから実行してください。
