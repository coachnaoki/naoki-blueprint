01環境セットアップ（初回のみ）

動画制作に必要な4ツールをインストールします。すでに入っている方はスキップしてOK。

macOS (Apple Silicon)
brew install node python@3.12 ffmpeg pip3.12 install mlx-whisper opencv-python npm install -g @anthropic-ai/claude-code
Windows

まずブラウザから事前準備:

  • Node.js: nodejs.org
  • Python 3.12: python.org（「Add Python to PATH」必須）
  • FFmpeg: gyan.dev/ffmpeg（PATHに追加）

その後 PowerShell で:

pip install faster-whisper opencv-python npm install -g @anthropic-ai/claude-code
⚠ Python は 3.12 限定。3.14 では mlx-whisper が動きません。

02リポジトリ取得

さん、準備を始めましょう。

⚠ 既に Claude Code を開いている方は、ターミナル上で Ctrl+C（Mac は Control+C）を 2回押して終了してから進めてください。Step 05 まで Claude Code は起動しません。

① naoki-blueprint を置きたいフォルダに移動（お好きな場所でOK。以下は例）:

cd ~/Desktop/Cursor

💡 ~/Desktop/Cursor は例です。すでに別のフォルダで作業している方はそちらに cdしてください。迷ったら ~/Desktop（デスクトップ直下）でも大丈夫。Finder で作成済みのフォルダなら何でもOK。

② リポジトリをダウンロード（Mac / Windows 共通）:

git clone https://github.com/coachnaoki/naoki-blueprint.git

③ ダウンロードしたフォルダに入る:

cd naoki-blueprint

📌 Step 03・04 も通常のターミナルのまま進めます。Claude Code は Step 05 でプロジェクトフォルダに入ってから起動します（起動タイミングを間違えると認証・カレントディレクトリがずれる原因になるため）。

03ライセンス認証

通常のターミナルでテンプレートフォルダに入り、あなたのライセンスIDを貼り付けて認証します:

cd .template node scripts/validateLicense.mjs NK-XXXX-XXXX-XXXX cd ..

✅ 認証成功で .license ファイルが生成され、以降は自動認証されます。PCは1台に紐付きます。

05プロジェクトに入って Claude Code 起動 → 素材配置 → 作業開始

✅ ここで初めて Claude Code を起動します。プロジェクトフォルダに入ってから起動することで、/step01-context が即動きます。

① プロジェクトフォルダに移動（先ほどの新規作成.sh で指定したプロジェクト名を使う）:

cd projects/2026-04-19

② Claude Code を起動（Mac / Windows 共通）:

claude --dangerously-skip-permissions

③ 素材を public/ に配置（Finder / エクスプローラーからドラッグ＆ドロップでOK）:

public/ ├── videos/ 動画素材 │ ├── main/ 本編動画 ★必須 │ ├── inserts/ 物理挿入動画（任意） │ ├── overlays/ オーバーレイ動画（任意） │ ├── opening/ OP動画（横動画のみ・任意） │ └── highlight/ ハイライト動画（自動生成・横動画のみ） ├── images/ 画像素材 │ ├── inserts/ 横に表示する画像（任意） │ └── overlays/ 全画面表示画像・アニメーション付（任意） ├── bgm/ BGM（MP3） ★必須 ├── se/ SE効果音 ★必須 │ ├── 強調/ │ ├── ポジティブ/ │ └── ネガティブ/ ├── script/ ナレーション台本（推奨） └── output/ 完成動画の出力先

④ 素材を置いたら、Claude Code で作業開始:

/step01-context

あとは AI が次のコマンドを案内してくれるので、質問に答えながら進めるだけ。

🎬ワークフロー詳細（横動画 全20 step / ショート 全14 step）

各ステップは Claude Code でスラッシュコマンド1つで実行できます。任意 とあるステップは必要な演出だけ使えばOK。基本は AI が次のコマンドを案内してくれるので、順番通りに進めるだけで完成します。

PHASE 1 / 素材準備
  • /step01-context 動画の目的・ターゲット・視聴者像を整理
  • /step02-assets 素材確認（本編/物理挿入/オーバーレイ/OP/ハイライト）とFPS自動検出
  • /step03-transcript 本編を Whisper で文字起こし（word-level timestamps）
  • /step04-transcript-fix 台本と照合して誤変換を修正（固有名詞・数字を重点チェック）
  • /step05-cut 無音 + 言い直しを一括カット。word-boundary snap 92%精度（v1.3.0）で発話プツッ切れを防止
  • /step06-transcript カット後の再文字起こし + 修正再適用
PHASE 2 / 動画構築
  • /step07-template テンプレート設定（フォント・サイズ・色・SE音量）を templateConfig.ts に定義
  • /step08-telop テロップデータ作成。AI が 8種のスタイル（normal/emphasis/negative等）から自動判定
  • /step09-composition MainComposition.tsx で動画 + テロップ + SE を統合。Remotion Studio がブラウザで起動（localhost:3000）
PHASE 3 / 素材・演出挿入（任意）
  • /step10-greenback 任意 グリーンバック背景置換。chromakey + rembg + HSV despill の3段パイプライン
  • /step11-videos 任意 デモ動画挿入（物理挿入=Series分割 / オーバーレイ=上に重ね）
  • /step12-slides-gen 任意・横のみ 台本からスライドHTMLを生成
  • /step13-slides 任意・横のみ スライドキャプチャ + タイムライン配置
  • /step14-wipe 任意・横のみ ワイプ（顔出し）位置をピクセル座標で調整
  • /step15-images 任意 画像挿入（3フェーズ構造: インサート / オーバーレイ / 顔アイコン）。Gemini AI画像生成対応
  • /step16-special-components 任意・横のみ 箇条書き・見出しバナー・CTAコンポーネント
  • /step17-endscreen 任意・横のみ エンドスクリーン（チャンネル登録誘導）追加
PHASE 4 / BGM・出力
  • /step18-bgm BGM挿入（フェードイン/アウト付き、区間・音量指定可、不足時は自動ループ）
  • /step19-opening 横のみ OP連結 + 本編レンダリング（1回目）
  • /step20-highlight-final 本編からハイライト自動抽出 + ラウドネス正規化 -14 LUFS（v1.3.0）+ 最終レンダリング → 完成 🎉
🎬 ショート動画（縦 / 14 step）との違い: スライド生成（step12-13）・ワイプ（step14）・特殊コンポーネント（step16）・エンドスクリーン（step17）・OP連結（step19）・ハイライト抽出（step20）は横動画のみ。ショートは必要最小限の14 step で情報密度を優先し、/step14-final で 1080×1920 レンダリングして完成します。
💡 便利コマンド: /catchup で現在の進捗を確認できます。中断しても AI がどのステップから再開すべきか教えてくれるので、気軽に休憩OK。

UPアップデート（完全自動・v1.5.5〜）

何もしなくてOK。最新版が自動で適用されます。

各 step 実行時に内部で 24時間に1回だけ 最新版チェックが走り、新しいバージョンがあれば自動でダウンロード・適用されます。作業中の素材（public/）・テロップデータ（src/*.tsx）・transcript は保護されるので、いつ自動更新が走っても作業は壊れません。

更新があると Claude Code のログに以下のようなメッセージが一瞬表示されます:

📦 naoki-blueprint v1.5.5 → v1.5.6 に自動更新中...
✓ 更新完了

※ どうしても今すぐ最新版にしたい場合（レアケース）: Claude Code を Ctrl+C を 2回 押して終了し、通常ターミナルで以下を実行:

cd ~/Desktop/Cursor/naoki-blueprint && bash アップデート.sh

通常は自動更新に任せてOKです。
