Naoki式 丸投げビジネス動画編集テンプレート

Claude Code を使って、撮影した動画から YouTube 横動画（全20ステップ）/ ショート動画（全14ステップ）を自動制作するツールキット

0 環境セットアップ（初回のみ）

動画制作に必要なツールをインストールします。既にインストール済みのものはスキップしてOK。

確認コマンドを実行して、バージョン番号が表示されればインストール済みです。

ツール何に使う？確認コマンドインストール方法
Node.js (v18+)動画の組み立てnode -vnodejs.org
FFmpeg動画のカット・結合ffmpeg -version下記参照
Python 3.12文字起こしpython3.12 --version下記参照
WhisperAI音声→テキスト変換下記参照下記参照
Claude CodeAI動画制作の自動化claude --versionclaude.ai

Mac（Apple Silicon）の場合

brew install node python@3.12 ffmpeg
pip3.12 install mlx-whisper opencv-python
npm install -g @anthropic-ai/claude-code
注意: mlx-whisper は Python 3.12 でのみ動作します（3.13以降では動きません）。

Windows の場合

まずブラウザから事前準備：

  • Node.js: nodejs.org
  • Python 3.12: python.org（「Add Python to PATH」にチェック必須）
  • FFmpeg: gyan.dev/ffmpeg（PATHに追加）

その後 PowerShell で以下を実行：

pip install faster-whisper opencv-python
npm install -g @anthropic-ai/claude-code
Whisperモデルの自動切替: macOSは mlx-whisper（Apple Silicon最適化で高速）、Windowsは faster-whisper を使用。精度は同等（どちらもlarge-v3モデル）。scripts/transcribe.mjs が自動で切り替えます。
OpenCV の用途: ワイプ（ピクチャーインピクチャー）の顔位置を自動検出するのに使います。

1 リポジトリをダウンロード

⚠ 事前準備: すでに Claude Code を開いている方は、ターミナル上で Ctrl+C（Mac は Control+C）を 2回押して終了してから進めてください。

Cursor のターミナルを開き、naoki-blueprint を置きたいフォルダに移動します（例: デスクトップの Cursor フォルダ）。

cd ~/Desktop/Cursor
git clone https://github.com/coachnaoki/naoki-blueprint.git
cd naoki-blueprint
以降のコマンドは Mac / Windows 共通です。Cursor のターミナルで、そのままコピペして実行してください。

2 Claude Code を立ち上げる

ここからは Claude Code に任せます。先に起動しておけば、以降のセットアップで詰まったときも Claude が代わりに解決してくれます。

claude --dangerously-skip-permissions
初回起動時にログインを求められた場合は、画面の指示に従ってブラウザで認証してください。

3 ライセンス認証（初回のみ）

権利者から発行されたライセンスIDを入力します。

cd .template
node scripts/validateLicense.mjs NK-XXXX-XXXX-XXXX
cd ..

認証成功すると .license ファイルが生成されます。2本目以降はこのステップは不要です（テンプレートをコピーすると .license も一緒にコピーされます）。

PCを変更する場合は Chatwork で Naoki に個別メッセージしてください。

4 プロジェクトフォルダを作成

動画1本ごとに専用のプロジェクトフォルダを作ります。作る動画の種類によってテンプレートが2つあります。

動画の種類テンプレート解像度ステップ数
YouTube横動画.template/1920×108020ステップ
ショート動画.template-shorts/1080×192014ステップ

おすすめは自動スクリプト。プロジェクト名と動画タイプを選ぶだけでフォルダ作成と npm install まで自動で終わります。

bash 新規作成.sh

起動すると以下を順に聞かれます：

  1. プロジェクト名（日付でもOK。例: 2026-04-19）
  2. 動画タイプ（1: YouTube横動画 / 2: ショート動画）
なぜ projects/ に作るの？
テンプレートやツール類はリポジトリ直下に残るので、2本目・3本目を作るときも同じ手順でコピーするだけ。

5 素材を配置

プロジェクトフォルダの public/ の中に、素材ごとの専用フォルダが用意されています。Finder（Mac）やエクスプローラー（Windows）で、各フォルダにファイルをドラッグ＆ドロップするだけでOKです。

public/
├── videos/                ← 動画素材
│   ├── main/              ← 本編動画（MP4） ★必須
│   ├── inserts/           ← 物理挿入動画（ナレーションを止めて挟む・任意）
│   ├── overlays/          ← オーバーレイ動画（ナレーション中に上に被せる・任意）
│   ├── opening/           ← OP動画（冒頭連結・横動画のみ・任意）
│   └── highlight/         ← ハイライト動画（自動生成・横動画のみ）
├── images/                ← 画像素材
│   ├── inserts/           ← 横に表示する画像（720×405・固定位置・任意）
│   └── overlays/          ← 全画面表示画像（アニメーション付き・任意）
├── bgm/                   ← BGM（MP3） ★必須
├── se/                    ← SE（効果音） ★必須
│   ├── 強調/              ← 通常の強調・箇条書き・表など
│   ├── ポジティブ/         ← 前向き・成功・インパクト系
│   └── ネガティブ/         ← 警告・失敗・ツッコミ系
├── script/                ← ナレーション台本（テキスト）
├── slides/                ← （自動生成されるので触らなくてOK）
└── output/                ← （完成動画の出力先）
素材必須？説明
本編動画必須撮影したMP4ファイルを videos/main/ に入れる
物理挿入動画任意ナレーションを止めて挟むクリップを videos/inserts/ に入れる
オーバーレイ動画任意ナレーション中に上に被せる補足映像を videos/overlays/ に入れる
OP動画任意冒頭のタイトル・ロゴ映像を videos/opening/ に入れる（横動画のみ）
ハイライト動画—完成後に自動生成（横動画のみ）
横挿入画像任意720×405 の画像を images/inserts/ に入れる
全画面画像任意全画面アニメーション表示用の画像を images/overlays/ に入れる
BGM必須フリーBGMサイト等からダウンロードしたMP3を bgm/ に入れる
SE（効果音）必須フリーSEサイト等からMP3をダウンロードして se/ に入れる
ナレーション台本推奨話した内容のテキストを script/ に入れる（文字起こしの精度UP）

6 動画制作スタート！

Step 2 で起動した Claude Code のタブに戻って、以下を入力してください：

/step01-context

あとはAIの指示に従って、/step02-assets → /step03-transcript → … と順番に進めるだけ！

アップデート（バージョンアップ）

テンプレートやスキルが更新されたら、以下のコマンドで最新版に置き換えできます。
あなたが作った projects/ と .license は安全です。

cd ~/Desktop/Cursor/naoki-blueprint
bash アップデート.sh
⚠ 注意: アップデート中は Claude Code を Ctrl+C を2回押して終了してから実行してください。

ワークフロー全体像 ― YouTube横動画（全20ステップ）

フェーズ1: 素材準備
/step01-context動画の目的・ターゲットを整理
/step02-assets素材確認＆役割確定（本編/物理挿入/オーバーレイ/OP/ハイライト）
/step03-transcript本編をWhisperで文字起こし
/step04-transcript-fix台本と照合して誤変換を修正
/step05-cut無音＋言い直し一括カット（FFmpeg一発エンコード）
/step06-transcriptカット後の再文字起こし＋修正再適用
↓
フェーズ2: 動画構築
/step07-templateテンプレート設定（フォント・サイズ・色・SE）
/step08-telopテロップデータ作成
/step09-compositionコンポジション統合（動画の骨格が完成）
↓
フェーズ3: 素材・演出挿入
/step10-greenbackグリーンバック背景置換 （任意）
/step11-videosデモ動画挿入（物理挿入=Series分割 / オーバーレイ=上に重ね） （任意）
/step12-slides-gen台本からスライドHTML生成 （任意）
/step13-slidesスライドキャプチャ＋タイムライン （任意）
/step14-wipeワイプ位置調整 （任意）
/step15-images画像挿入（感情ベース + AI画像生成） （任意）
/step16-special-components箇条書き・見出し・CTA （任意）
/step17-endscreenエンドスクリーン追加 （任意）
↓
フェーズ4: BGM・冒頭連結・出力
/step18-bgmBGM挿入
/step19-openingOP連結＋本編レンダリング（1回目・仮出し）
/step20-highlight-final本編MP4からハイライト自動抽出＋最終レンダリング → 完成！

ワークフロー全体像 ― ショート動画（全14ステップ・縦1080×1920）

フェーズ1: 素材準備
/step01-context動画コンテキスト整理（縦動画指定）
/step02-assets素材確認＆役割確定
/step03-transcript本編をWhisperで文字起こし（large-v3固定）
/step04-transcript-fix台本と照合して誤変換を修正
/step05-cut無音＋言い直し一括カット
/step06-transcriptカット後の再文字起こし＋修正再適用
↓
フェーズ2: 動画構築
/step07-templateテンプレート設定（8種のテロップ）
/step08-telopテロップデータ作成（台本一字一句）
/step09-compositionコンポジション構築（1080×1920）
↓
フェーズ3: 素材挿入
/step10-greenbackグリーンバック背景置換 （任意）
/step11-videosデモ動画挿入 （任意）
/step12-images画像挿入（AI生成・話者アイコン） （任意）
↓
フェーズ4: BGM・出力
/step13-bgmBGM挿入（音量0.03推奨）
/step14-final最終レンダリング（1080×1920 MP4）→ 完成！

naoki-blueprint/                 ← ダウンロードしたフォルダ
├── README.md                    ← ガイド
├── 新規作成.sh                   ← プロジェクト作成スクリプト
├── アップデート.sh               ← 最新版取得スクリプト
├── .template/           ← YouTube横動画テンプレート（編集しない）
│   ├── .claude/skills/          ← 20個のAIスキル + catchup + remotion-best-practices
│   ├── CLAUDE.md                ← AIの行動ルール
│   ├── scripts/validateLicense.mjs  ← ライセンス認証
│   ├── public/videos/main/      ← 本編動画
│   ├── public/videos/inserts/   ← 物理挿入動画（任意）
│   ├── public/videos/overlays/  ← オーバーレイ動画（任意）
│   ├── public/videos/opening/   ← OP動画（任意）
│   ├── public/videos/highlight/ ← ハイライト（自動生成）
│   ├── public/images/inserts/   ← 横挿入画像（任意）
│   ├── public/images/overlays/  ← 全画面画像（任意）
│   ├── public/bgm/              ← BGM
│   ├── public/se/               ← SE素材
│   ├── public/script/           ← ナレーション台本
│   └── ...
├── .template-shorts/    ← ショート動画テンプレート（編集しない）
│   └── .claude/skills/          ← 14個のAIスキル
├── aislides/                    ← スライドテンプレートシステム（15種類）
│
└── projects/                    ← あなたの動画プロジェクトはここ
    ├── tennis-serve-tips/       ← 例：1本目
    ├── cooking-pasta/           ← 例：2本目
    └── ...                      ← 何本でもOK
