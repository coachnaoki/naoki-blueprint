# My Style — 自分の話し方・好みのスタイル

このファイルは Claude Code が台本・テロップ・構成を生成する際に参照します。
**自分のクセ・好み・NGを具体的に書くほど、生成物が自分らしくなります。**

`my-style.example.md` をコピーして `my-style.md` にリネームし、自分用に書き換えてください。

---

## 基本プロフィール

- **名前 / チャンネル名**: （例: 田中太郎 / たなかチャンネル）
- **発信ジャンル**: （例: テニス指導 / AI活用 / 料理 / 投資）
- **想定視聴者**: （例: 40〜60代テニス中級者 / 副業で月5万稼ぎたい初心者）
- **ポジショニング**: （例: 「論理で教える唯一のコーチ」「現役エンジニアが教える〜」）

---

## 話し方のクセ

### 一人称・二人称
- **一人称**: （例: 「僕」「私」「俺」）
- **二人称**: （例: 「あなた」「みなさん」「君」）
- **呼びかけ方**: （例: 「◯◯したい方へ」「本気の方だけ聞いてください」）

### 口癖・決めゼリフ
- （例: 「はっきり言います」）
- （例: 「3秒で考えてみてください」）
- （例: 「絶対に大丈夫」）

### 文体の好み
- 文末の比率: （例: 断定60% / 問いかけ20% / 共感10% / 行動喚起10%）
- 一文の長さ: （例: 30〜50文字、スマホ読みやすさ重視）
- 体言止め: （例: キーワード強調のため多用する）

---

## NGワード・使わない表現

Claude Code に **絶対に使わせたくない言葉・表現** を書いてください。

- （例: 「〜と思います」の連発は避ける。60%以上は「〜です」で断言）
- （例: 感情論・根性論は禁止。論理・数値・物理で説明する）
- （例: 「すごい」「ヤバい」は使わない）
- （例: 英語のカタカナ略語は避ける: コミット→約束、レバレッジ→てこ）

## 得意ジャンル・定番テーマ

自分が扱うことが多いテーマ。Claude Code が構成を考える際のヒントになります。

- （例: ダブルス戦術 / シングルス戦術 / メンタル強化）
- （例: Claude Code 活用 / GAS自動化 / AI動画編集）

## 好みのテロップ・BGM・演出

### テロップ
- 強調の仕方: （例: 重要キーワードは黄色背景＋黒文字）
- テロップ量: （例: ほぼ全発話を入れる / キーワードだけ）
- 絵文字: （例: 使わない / 要所で使う）

### BGM
- トーン: （例: 穏やか / 盛り上がる / 感情系）
- 避けるジャンル: （例: EDM / ロック）

### 演出
- ワイプ位置の好み: （例: 右下固定 / 動的）
- SE: （例: 控えめ / しっかり効かす）
- 間の長さ: （例: テンポ重視で短め / 考える時間を取って長め）
